# Spam Filter
A basic linear support vector machine classifier for spam emails detection.
## Dependencies
+ OS
+ csv
+ string
+ numpy
+ pandas
+ collections
+ nltk
+ sklearn
## Data Representation
+ Bag of words
+ TF-IDF
## Usage Instructions
Simply run:
```
python train.py
```
+ Loading raw data set
+ Word embedding
+ Preprocessing
+ Training

## Score:
+ The accuracy score is in 0.985845 - 0.996347 based on cross validation.
